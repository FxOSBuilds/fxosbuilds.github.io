Testing

We need your help to test the install scripts and the builds in order to deliver a great experience to users.

How can I help?

If you have one of our supported devices, please, download the install script and follow the install guide instructions. If something goes wrong, report it back in our github page, don't forget to detail:

  • Your device model.
  • Your computer OS.
  • Your previous Firefox OS version.
  • The Firefox OS version your are trying to update to.
  • If you have modified the factory install (installing other builds previously, rooting...).
  • Messages you got on screen during install.

Also, if everything goes right, let us know! We would like to know about your success, write us on our Twitter account.

Thanks, your help will allow us to improve and give a better experience to other users like you :)
