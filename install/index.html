Install

Requisites

Your system need these tools installed first in order to run the update utility.

    GNU/Linux
  • perl (pre-installed with the OS)
  • gzip
  • java (openjdk-7-jre package)
  • adb (android-tools-adb package)
    Windows
  • perl
  • java
  • adb
    MacOS
  • perl (pre-installed with the OS)
  • java
  • adb

Step 1

Download the package for your device from our font page and unzip it.

Step 2

On your device, enable the remote debugging. You will find it under Settings - Device information - More information - Developer - Remote debugging.

Step 3

Connect your device via USB cable and, on your computer, open a terminal inside the folder you've just unziped and run:

GNU/Linux: ./flash_linux.sh

Windows: ./flash_windows.bat

Mac: ./flash_mac.sh

Important note: If your device wasn't 'rooted' before (if you don't know what this mean, it was not) you have to select first the option "Root your device".

Once your device is rooted, run the script and select the "Update your device" option. Use enter to accept the options.

The device will reboot itself a few times, follow the instructions in screen.

If you find any issue with the installed (not with the build), please report it on our github page.

Enjoy!
